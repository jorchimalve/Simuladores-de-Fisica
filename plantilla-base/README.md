# Plantilla base

Carpeta de referencia para crear nuevos simuladores con la identidad visual del proyecto.

Flujo sugerido:

1. Copiar esta carpeta.
2. Pegar la copia dentro de `simuladores/`.
3. Renombrar la carpeta con un identificador corto, en minúsculas y sin espacios.
4. Mantener `index.html` como archivo principal.
5. Usar `assets/css/simulator-base.css` y `assets/js/simulator-theme.js` para conservar el estilo común.
