# Plantilla base

Usa esta carpeta como punto de partida para crear nuevos simuladores.

Flujo sugerido:

1. Copia `plantilla-base/`.
2. Pega la copia dentro de `simuladores/`.
3. Cambia el nombre de la carpeta por un identificador corto, en minúsculas y sin espacios.
4. Mantén el archivo principal como `index.html`.
